# DGT_cita_previa

Un _script_ rápido para ayudarte a reservar cita en la DGT (España).
Comprueba las citas disponibles periódicamente, y manda un mensaje
cuando haya un hueco disponible. La reserva se hace entonces
manualmente.

*Actualización*: Hoy en día, la web de la DGT usa un CAPTCHA para evitar
que robots como éste automaticen la búsqueda de citas. Por lo tanto,
este _script_ ya no funciona.

python 3.8
Requiere selenium webdriver (por ejemplo: chromedriver.exe en la misma
carpeta)
