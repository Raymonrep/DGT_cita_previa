# DGT_cita_previa

Un script rápido para ayudarte a reservar cita en la DGT (España).
Comprueba las citas disponibles periódicamente, y manda un mensaje
cuando haya un hueco disponible. La reserva se hace entonces
manualmente.

python 3.8
Requiere selenium webdriver (por ejemplo: chromedriver.exe en la misma
carpeta)
